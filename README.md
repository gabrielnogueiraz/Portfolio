## Como executar o projeto

### Pré-requisitos

- Node.js
- npm ou yarn

### Instalação

1. Clone o repositório
```bash
git clone https://github.com/seu-usuario/gabriel-portfolio.git
cd gabriel-portfolio
```

2. Instale as dependências
```bash
npm install
# ou
yarn install
```

3. Execute o projeto em modo de desenvolvimento
```bash
npm start
# ou
yarn start
```

4. Acesse http://localhost:3000 para ver o site em funcionamento

### Build para produção

```bash
npm run build
# ou
yarn build
```
